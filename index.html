ABOUT ME

Etonix Pyro is a creative, gigantic man-child unleashed upon the world with a rather uncanny weapon - a keyboard! Wielding this noisy slab of clicks, he skillfully weaves strange symbols into masterpieces. Based in the United States with over fifteen years of experience in the industry, when not working as a freelance developer or consultant, he can be found working on assets for the Unity Asset Store. Etonix Pyro is the founder of Shadowed Souls Studios, an independent game studio and consulting agency.

MY RECENT PROJECTS

UMA for RPG Builder

UMA for RPG Builder is a plugin for RPG Builder that allows users to use UMA 2.0 characters in their projects with ease.

ShadowBox: Game Toolkit

ShadowBox: Game Toolkit is the successor to ShadowBox: Interactivity Toolkit. It is modular in design, with each module adding a new feature to the toolkit. The base product will include the following modules:

  • Archivist, a save/load module for character and game data.
  • Interactions, an interactivity module for game objects.
  • Magellan, a minimap, compass, and location marker module for player guidance.
  • Stagehand, a scene management and loading screen module
  • ... and many more to come!

It is currently in development and will be released soon.

Asset Invoice Verification Bot

A Discord bot that grants roles to members of the server that provide a valid invoice from the Unity Asset Store or Shopify, and optional GitHub Repository access per package. Additionally, any messages containing hyperlinks for the Unity Asset Store or Humble Bundle are automatically filtered and restored with appropriate affiliate marketing data. This is a managed service currently available to Unity Asset Store Publishers. Also available without Discord integration as a web service.
